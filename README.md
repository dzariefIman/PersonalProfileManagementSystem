## Description

A profile management system using Java Servlet, JSP, JDBC, and Derby database. Users can add, view, search, edit, and delete student profiles.

## Features

- Add Profile
- View Profiles
- Search Profile (by ID or name)
- Edit Profile
- Delete Profile
